# Corefinity Hosting Control Panel

A modern, responsive web-based hosting control panel with a sleek dark theme UI, built with vanilla HTML, CSS, and JavaScript.

## 🚀 Features

### Core Functionality
- **Dashboard**: Overview of system status, resource usage, and quick actions
- **Environment Management**: Create, configure, and manage hosting environments
- **Deployment Management**: Track and manage application deployments
- **Support System**: Comprehensive ticketing system for customer support
- **User Profile**: Account settings and preferences management

### UI/UX Highlights
- **Modern Dark Theme**: Professional navy-based color scheme with vibrant accent colors
- **Responsive Design**: Fully responsive layout that works on all devices
- **Interactive Components**: Smooth animations, hover effects, and transitions
- **Accessibility**: Semantic HTML5 structure with proper ARIA labels
- **Performance**: Optimized CSS and minimal JavaScript for fast loading

## 📁 Project Structure

```
Corefinity Hosting Control Panel/
├── index.html          # Main dashboard page
├── environments.html    # Environment management interface
├── deployment.html     # Deployment tracking and management
├── tickets.html        # Support ticket listing and management
├── ticket-view.html    # Individual ticket view and response
├── profile.html        # User profile and settings
└── README.md          # This documentation file
```

## 🎨 Design System

### Color Palette
- **Primary Navy**: `#080f1a` (background)
- **Card Navy**: `#111e2e` (card backgrounds)
- **Accent Blue**: `#2196ff` (primary actions)
- **Accent Orange**: `#f5722a` (secondary actions)
- **Success Green**: `#22d45e` (success states)
- **Warning Yellow**: `#f5a623` (warning states)
- **Danger Red**: `#ef4444` (error states)

### Typography
- **Display Font**: Syne (modern, geometric)
- **Body Font**: DM Sans (readable, friendly)
- **Monospace Font**: JetBrains Mono (code/technical data)

### Layout Features
- **Border Radius**: 14px (cards), 9px (small elements), 20px (large elements)
- **Shadows**: Multi-layered shadow system for depth
- **Transitions**: Smooth cubic-bezier animations (0.2s)

## 🛠️ Technology Stack

- **Frontend**: Pure HTML5, CSS3, and Vanilla JavaScript
- **Fonts**: Google Fonts (Syne, DM Sans, JetBrains Mono)
- **Icons**: Custom SVG icons and icon fonts
- **Design**: CSS Grid and Flexbox for responsive layouts
- **Theme**: CSS custom properties for consistent theming
